## Database Setup

location `.env` in `folderproject` config your database environment for `production or development`
```env
db_name = yournamedb
db_pass =
db_user = root
db_type = mysql
db_host = 127.0.0.1
db_port = 3306
token_password = thisIsTheJwtSecretPassword //Do not commit to git!

PORT_MACHINE = 9000
```
